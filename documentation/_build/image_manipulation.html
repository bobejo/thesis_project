image_manipulation¶

image_manipulation.binary_image(img, threshold)¶

Set all parameters below the threshold to 0 and all above to 1

Parameters:
  • img – The image
  • threshold – The threshold
Returns:

An binary image where each pixel is either 1 or 0
image_manipulation.crop_images(path)¶

Takes multiple images and crops to each blue box. Cropping sizes are chosen manually

Parameters:path – The path to the folder where the images are
image_manipulation.dilate_image(img, size)¶

Dilates the binary image, making the lines thicker and more connected

Parameters:
  • img – Binary image
  • size – Size of the dilation
Returns:

Dilated binary image
image_manipulation.image_segmentation(img, threshold, size)¶

Does image segmentation on the input image. First converts to binary image then dilates it

Parameters:
  • img – The output from prediction
  • threshold – The threshold for choosing 1 and 0
  • size – The size of the dilation
Returns:

A binary dilated image of type uint8 (wanted by blobdetector)
