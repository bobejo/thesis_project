img_numpy¶

img_numpy.imgs2numpy(imgs_path, maximum_imgs)¶

Reads all the images of a path and returns an numpy array of the images. Will take a lot of memory therefore one can use the maximum_imgs to specify how many images you want.

Param:The path to the images
Param:The maximum amount of images wanted
Returns:The numpy array of size
