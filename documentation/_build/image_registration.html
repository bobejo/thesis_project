image_registration¶

image_registration.affine_transformation(A, t, lpoints)¶

Calculates the coordinates of the right image using affine transformation rpoints=A*lpoints+t

Parameters:
  • A – Transformation matrix (2x2)
  • t – Translation vector
  • lpoints – Points from left image
Returns:

The corresponding points in the right image
image_registration.affine_transformation_solver(lpoints, rpoints)¶

Calculates the transformation matrix and translation vector with 3 coordinates. theta=M

Parameters:
  • lpoints – List of coordinates (x,y), of at least length 3, for left camera
  • rpoints – List of coordinates (x,y), of at least length 3, for right camera
Returns:

Transformation matrix A and translation vector t
image_registration.camera_transform(T, U_camera)¶

Takes camera points and transform them to the UR10 base frame.

  • T – Transformation matrix (3x4)
  • U – Triangulated points in camera frame (X;Y;Z)

image_registration.get_residuals(A, t, lpoints, rpoints)¶

Estimates the residuals for the transformation

Parameters:
  • A – Transformation matrix
  • t – Translation vector
  • lpoints – Points from left image that were used for calculating A and t
  • rpoints – Points from right image
Returns:

The absolute residual
image_registration.least_square_solver(lpoints, rpoints, threshold)¶

Uses the least square method the estimate the transformation matrix and translation vector. First removes outliers using ransac_inliers

Parameters:
  • lpoints – List of coordinates (x,y), of at least length 3, for left camera
  • rpoints – List of coordinates (x,y), of at least length 3, for right camera
  • threshold – The treshold used for removing outliers in RANSAC
Returns:

Transformation matrix A and translation vector t
image_registration.ransac_inliers(lpoints, rpoints, threshold)¶

Takes matched points from feature_match, does affine transformation and removes points which gives outliers.

Parameters:
  • lpoints – Left image features from feature_matching
  • rpoints – Right image features from feature_matching
  • threshold – The threshold for classifying points as outliers
Returns:

The inliers for left and right points
